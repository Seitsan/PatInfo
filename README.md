# PatInfo

Десктопное приложение для управления медицинскими данными пациентов с возможностью расчета индекса массы тела (ИМТ) и визуализации статистики.

## Возможности

- Управление пациентами: Добавление, редактирование, удаление записей
- Расчет ИМТ: Автоматический расчет индекса массы тела и категоризация
- Визуализация статистики: Графики распределения по полу, возрасту, ИМТ
- Валидация данных: Проверка корректности вводимых данных
- Сохранение данных: Автосохранение в JSON формате

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/Seitsan/PatInfo.git
cd PatInfo

# Создание окружения
conda env create -f environment.yml
conda activate patinfo_venv
```

## Запуск
```bash

python main.py
```
## Функциональность

### Основные модули

#### Главное окно (MainWindow)

- Таблица со списком пациентов
- Кнопки управления (Добавить пациента, Редактировать, Удалить)
- Двойной клик для редактирования

#### Форма пациента (PatientForm)

- Ввод данных пациента
- Валидация полей
- Редактирование существующих записей

#### Статистика (StatisticsWindow)

- Круговая диаграмма распределения пациентов по полу
- Гистограмма распределения пациентов по возрасту
- Box Plot распределения ИМТ по полу
- Scatter Plot распределения ИМТ по возрасту

#### Менеджер данных (DataManager)

- Загрузка/сохранение данных в JSON
- Генерация демонстрационных данных
- Обработка ошибок чтения/записи

### Модель данных

#### class Patient

- Основные данные (ФИО, пол, дата рождения)
- Антропометрические данные (рост, вес)
- Вычисляемые свойства (возраст, ИМТ, категория ИМТ)
